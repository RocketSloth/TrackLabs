About Lab Result Tracker

Lab Result Tracker is a web application that allows users to track their lab results over time. Users can input their lab results from doctor's visits and the application will keep track of each entry, analyze all the entries, and alert the user of any concerning trends.

The application is built using React and uses Supabase for data storage and user authentication.
